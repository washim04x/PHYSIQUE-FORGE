FAQ

The app is fully static. If you want a smaller or simpler deployment later, the logic can be trimmed without changing the visual style.

Will this run offline if I open the HTML file directly?

Yes. It is built for the browser only and does not require a backend, Node.js, or a database.

Can I host it on free static hosting?

Yes. It is one HTML file with embedded CSS and JavaScript, so it works well on GitHub Pages, Netlify, and similar hosts.

Does it really use the nutrition and exercise data?

Yes. The generator embeds a selective nutrition database and a selective exercise library pulled from your provided research documents.

Can you make it even lighter later?

Yes. We can remove sections, reduce the exercise library, or simplify the plan rendering while keeping the same design language.
